# Pocket Watcher
## Description
Pocket Watcher is a helpful money app that makes it easy to understand and control your cash. It helps you remember your bills, see what you spend, set saving goals, and watch how close you are to reaching them. It turns budgeting into a simple habit instead of a hard chore. With AI analysis and planing financial freedom is only a click away.

## Features
* User Profile
* Personal Bank Data
* AI analysis and planing
* Personal Dashboard with Data
* Spending Summary
